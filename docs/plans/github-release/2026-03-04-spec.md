# Spec: Automated GitHub Release + Install Script

Date: 2026-03-04

## Goal

Automate binary releases for macOS and provide a one-liner install script so users can install `disk-tree` without Rust toolchain.

## Deliverables

1. `.github/workflows/release.yml` — CI release workflow
2. `install.sh` — curl-based install script
3. Updated `README.md` — install one-liner at the top

---

## 1. Release Workflow (`.github/workflows/release.yml`)

### Triggers

- `push` on tags matching `v*` (e.g. `v1.0.0`)
- `workflow_dispatch` with a required `version` input (e.g. `v1.0.0`)

### Jobs

**build** — matrix over two targets:
- `x86_64-apple-darwin`
- `aarch64-apple-darwin`

Steps per target:
1. `actions/checkout`
2. Install Rust stable + target via `rustup target add`
3. `cargo build --release --target <target>`
4. Rename binary to `disk-tree-<target>`
5. Upload artifact

**release** — depends on `build`
1. Download both artifacts
2. Create GitHub Release (tag = pushed tag or dispatch input)
3. Attach both binaries as release assets

### Notes
- Use `actions/upload-artifact` / `actions/download-artifact` v4
- Use `softproj/action-gh-release` or `gh release create` CLI for release creation
- Strip binaries before upload to reduce size

---

## 2. Install Script (`install.sh`)

```
detect arch (uname -m) → arm64 or x86_64
fetch latest release tag from GitHub API
construct download URL: .../releases/download/<tag>/disk-tree-<arch>-apple-darwin
download to /tmp/disk-tree
chmod +x, move to /usr/local/bin/disk-tree
print success message
```

Error handling:
- Fail fast with `set -e`
- Print clear error if unsupported arch
- Print clear error if download fails

---

## 3. README Update

Add at the top of the README, before Features:

```markdown
## Install

```sh
curl -fsSL https://raw.githubusercontent.com/<user>/disk-tree/main/install.sh | sh
```
```

---

## Success Criteria

- Pushing `git tag v1.0.0 && git push --tags` produces a GitHub Release with two binary assets
- Running the curl install one-liner on an M-series Mac installs the correct binary
- Running the curl install one-liner on an Intel Mac installs the correct binary
